## Git Commands
- git clone
- git init
- git add
- git commit
- git push
- git status
- git log
- git pull
- git remote
- git branch

| No| Git Commands | Elaboration |
|---|--------------|-------------|
| 1 | Git Clone    | It is used to copy a remote repository from a machine.|
| 2 | Git init |  It is used to create a newgit repository in the current directory.|
| 3 | Git add | It will add the files from working directory to staging area.|
| 4 | Git commit | It is used to save staged changes to the repository with a commit message.|
| 5 | Git push | Used to upload your local commits to a remote repository.|
| 6 | Git status | It is used to check the current state of your repository.|
| 7 | Git log | Used to see the commit history of a rpository.|
| 8 | Git pull | It brings new updates from the server to your computer.|
| 9 | Git remote | Used to connect local repo to the Github.|
| 10 | Git branch | By using this command we can create,view,rename and delete branches in Git.|
